# Fonts Directory

Place your self-hosted font files here:

- **ClashDisplay-Variable.woff2** — [Indian Type Foundry](https://www.indiantypefoundry.com/) (commercial license required)
- **CabinetGrotesk-Variable.woff2** — [Klim Type Foundry](https://klim.co.nz/) (free for personal use, commercial license required)

## Fallback Behavior

If these files are missing, the theme gracefully falls back to:
- **Clash Display** → Inter (loaded from Google Fonts CDN via theme.json fontFace)
- **Cabinet Grotesk** → Inter (loaded from Google Fonts CDN via theme.json fontFace)

## How to Add Fonts

1. Purchase the license from the foundry
2. Download the variable font (.woff2)
3. Place the file in this directory with the exact name listed above
4. Clear any caching plugins
